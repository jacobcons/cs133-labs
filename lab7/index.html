Lab 7: LaTeX Exercise

Exercises:

  1. Create a BibTeX file with several entries of a different type
  2. Cite said entries in a LaTex file and produce a pdf
  3. Download and reference the IEEE style for the document and bibliography
  4. Use the listings package along with custom settings to format the styles of some python

Commands:

  • Cite a BibTeX entry with the given label
    \cite{label}
  • Style the bibliography with the given style
    \bibliographystyle{style}
  • Use entries from the bibliography with the given file name
    \bibliography{file_name}
  • Generate bibliography information in a readable format
    bibtex $FILE_NAME
