Lab 6: LaTeX

Exercises:

  1. Create and compile a basic latex document
  2. Create a document using the article document class, 12pt font and two columns. Then create three sections divided into sub and subsub sections with some content
  3. Extend the document by adding some itemised lists, enumerated lists and figures of different placement specifiers and width
  4. Create a new document containing an algorithm to print all even numbers from 0 to n with caption and label

Commands:

  • Define the document class and global styles
    \documentclass[styles]{class}
  • Sectioning content
    \chapter{title}
    \section{title}
    \subsection{title}
    \subsubsection{title}
  • Table of contents
    \tableofcontents
  • Appendix
    \appendix
  • Numbered list 
    	
    
\begin{enumerate}
\item name
\end{enumerate}
  • Bullet point list 
    	
    
\begin{itemize}
\item name
\end{itemize}
  • Include graphic
    \includegraphics[width=value]{Path/To/File.png}
