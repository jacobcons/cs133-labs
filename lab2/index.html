Lab 2: Web Standards and Version Control Software

Exercises:

  1. Initalise a git repo
  2. Add, commit and push an html file
  3. Check commit status
  4. Change said file such that it's valid, and validate it using an online validator
  5. Check the difference between the last commit and current code
  6. Link to an external css file and write some styles for the title
  7. Add some divs with different id's and style them (absolute/relative positioning)

Commands:

  • View the changes in code since last commit
    git diff
