Lab 4: Advanced Scripting and Regular Expressions

Exercises:

  1. Write a program which prints the nth fibonacci number, making use of functions
  2. Redirect the output of a command into a textfile and vice versa
  3. Experiment with piping the output of a command into various filter commands
  4. Use grep to print all lines that contain a number using regular expressions
  5. Use grep on a dict file to print all words where the first 2 letters are vowels and it ends with 'ing'
  6. Use sed to print the sum of all the given arguments to a shell script
  7. Use sed to extract all words beginning with 'w' from a webpage
  8. Use grep to print all files which are world readable in a directory

Commands:

  • Prints all lines matching the given pattern
    grep $PATTERN $FILE
  • Print the first 10 lines of a file
    head $FILE
  • Replace target with given replacement in an input file
    sed 's/$TARGET/$REPLACEMENT/' $INPUT-FILE
