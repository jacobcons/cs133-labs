Lab 3: Linux Processes and Scripting

Exercises:

  1. Find the PID of a named daemon by piping the output of ps to grep
  2. Investigate various ways to kill a process; namely using kill, pkill and xkill
  3. Create a script called runloop.sh which compiles Loop.java and then runs it. The process was killed using pkill runloop.sh
  4. Extend the script to run the java program, sleep for 5 seconds and then kill it
  5. Write a script that multiplies two arguments together and outputs the result
  6. The loop works by iterating over all the arguments and storing each one in a variable called "a" for a given cycle. This is similar to the enhanced for-loop in java.
  7. Use a case statment within said loop to check each argument ensuring that it's a number. Also include a case for a help flag
  8. Write a script which sums arguments in pairs and outputs the results

Commands:

  • Evaluate a mathematical expression
    expr $EXPRESSION
  • Kill the process with the given PID
    kill -9 $PID
  • Kill all processes containing the given name
    pkill $NAME
  • Prompts user to select window of an application and kills all processes associated with said application
    xkill
